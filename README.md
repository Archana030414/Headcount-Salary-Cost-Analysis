# Headcount-Salary-Cost-Analysis
Workforce cost analysis evaluating headcount distribution and salary expenses to identify cost drivers and support resource planning.

### Tools Used
Microsoft Excel, Pivot Tables, Cost Analysis

### Analysis Performed
- Calculated total headcount and salary cost
- Derived average cost per employee
- Compared departmental salary distribution
- Visualized cost concentration areas

### Key Insights
* Salary costs concentrated in IT and Operations functions  
* Average employee cost indicates workforce cost efficiency  
* Headcount distribution highlights operational priorities  

### Business Recommendations
Monitor workforce costs regularly and align hiring decisions with budget planning and operational needs.
